# AthletIQ Validation Dataset

This folder is for the prototype validation workflow.

Use `source_log.csv` for every public candidate before downloading or testing it. Public video is only usable when the specific asset license is verified and the clip clearly shows the full approach, takeoff, peak jump, and landing.

Use `labels.csv` for one manually reviewed row per clip. Label frame numbers for:

- approach start
- penultimate step
- plant step
- takeoff
- peak jump
- landing

First real milestone: 10 clean first-party side-view clips with athlete consent, visible feet, and manual frame labels.
